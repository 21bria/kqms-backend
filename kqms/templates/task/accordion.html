Berisi path Python lengkap menuju fungsi Celery yang akan dijalankan untuk task tersebut.

Harus dalam format: package.module.function_name

Path ini akan divalidasi otomatis:
- Cek apakah path bisa di-import dengan import_string()
- Cek apakah fungsi tersebut memang ada di dalam modul

Jika salah, akan muncul pesan error seperti:
"Task path tidak valid: Module 'kqms.task' does not define a 'import_assay_mral' attribute"

Contoh Penulisan yang Benar:
kqms.task.import_assay_mral.import_assay_mral atau kqms.task.file_path.class
